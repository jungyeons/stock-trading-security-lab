# A05 Security Misconfiguration

## 학습 목표
- 기본 설정 실수(CORS, Swagger, 에러노출)를 점검한다.

## 위험 설명
- 과도한 CORS 허용, 공개된 관리 문서, 상세 스택트레이스 노출은 공격 표면을 넓힌다.

## 점검 포인트
- CORS 허용 도메인
- Swagger 접근 통제
- 에러 응답 포맷

## secure 대응
- 로컬/개발 환경 허용 범위 제한
- 관리자 또는 인증 사용자에게만 문서 노출

## 수정 후 검증 방법
- 비인증 Swagger 접근 차단 확인
- 에러 응답에 내부 구현 정보 미노출 확인
