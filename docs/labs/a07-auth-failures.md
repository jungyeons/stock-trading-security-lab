# A07 Identification and Authentication Failures

## 학습 목표
- 인증 실패 처리와 세션/토큰 통제를 설계한다.

## 위험 설명
- 로그인 실패 제한 부재, 토큰 만료 미적용은 계정 탈취 확률을 높인다.

## 점검 포인트
- 로그인 실패 감사 로그
- 토큰 생성/검증/만료 처리

## secure 대응
- 표준 인증 플로우 + 만료 토큰 거부
- 로그아웃 시 클라이언트 토큰 폐기 정책 안내

## 수정 후 검증 방법
- 잘못된 자격증명 시 실패 로그 생성 확인
- 만료 토큰으로 API 호출 시 `401` 확인
